# VideoGameCollectionDMS
A Java command-line application to manage and organize a personal video game collection with add, update, delete, bulk import, and inflation-adjusted pricing features.
The Video Game Collection DMS is a Java-based command-line application that allows users to manage a personal video game library. It provides features to add, delete, and update games with details such as ID, title, genre, platform, release year, price, and multiplayer status, as well as calculate inflation-adjusted prices. Users can also load games in bulk from a text file while the system validates input to prevent blank entries, duplicate IDs, and invalid values. This project offers a simple, menu-driven interface for organizing and maintaining a comprehensive collection of video games.
